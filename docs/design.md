# 设计系统主题

本文档概述了我们设计系统的核心美学和功能原则。

## 品牌色彩

我们的品牌采用充满活力且协调的调色板，以确保一致且可识别的用户体验。

- **主色 (Primary Color):** `#1A40C2` - 这是品牌最具辨识度的颜色，用于主要操作按钮、活跃状态和关键交互元素。
- **次色 (Secondary Color):** `#6277a3` - 辅助颜色，用于次要 UI 元素、次要操作和微妙装饰。
- **第三色 (Tertiary Color):** `#d06e00` - 额外的强调色，用于高亮、徽章或装饰元素，提供视觉趣味性。
- **中性色 (Neutral Color):** `#75777f` - 作为背景、表面和非色彩元素的基础，提供干净稳定的基石。

系统采用 `light` 浅色模式，确保最佳可读性和明亮、友好的界面。

## 排版系统

我们的排版系统建立在清晰度、可读性和品牌一致性之上。

- **标题 (Headlines):** `inter` - 用于所有主要标题，提供强烈的视觉层次。
- **正文 (Body Text):** `inter` - 应用于所有段落文本和一般内容，确保在各种屏幕尺寸上的可读性。
- **标签 (Labels):** `inter` - 用于表单标签、按钮和其他交互元素，确保清晰度和一致性。

## 形状与形态

UI 元素的几何特征决定了系统的整体感觉。

- **圆角 (Roundedness):** `1` - 我们的元素采用微妙的圆角，提供柔和而现代的美学。

## 间距

精心应用的间距确保平衡且通透的布局。

- **间距 (Spacing):** `2` - 我们的系统使用常规间距，在紧凑性和开放感之间取得平衡。
- **主容器内边距 (Main Container Padding):** `16px` - 所有主元素 (`<main>`) 和内容容器应在所有边保持一致的 16px 内边距，以确保适当的呼吸空间和视觉一致性。
- **列表页内边距 (List Page Padding):** `24px` - 列表页（需求、案例等）在所有边使用 24px 内边距，以实现最佳的内容密度和可读性。
- **列表页垂直间距 (List Page Vertical Spacing):** `12px` - 列表页中各部分之间的垂直间距应为 12px，以保持一致的节奏和层次。
